# ADR-003: 檢索策略選擇

## 背景
系統需要在軍事戰術知識檢索中，兼顧語義理解與精準度。  
因此必須選擇合適的檢索策略，確保能處理長文本並避免資訊遺漏。

## 選項
- **Keyword Search**：傳統關鍵字檢索，速度快，但語義理解不足。  
- **Semantic Search (純向量檢索)**：能捕捉語義，但可能忽略關鍵字精準度。  
- **Hybrid Search**：結合關鍵字與語義檢索，兼顧精準度與語義理解。  
- **Parent-Document Retrieval**：保留原始段落結構，適合長戰術文件，避免切分過度。  

## 決策
選擇 **Hybrid Search + Parent-Document Retrieval** 作為系統的檢索策略。  
理由：  
- Hybrid Search 能同時利用語義與關鍵字，提升查詢準確度。  
- Parent-Document Retrieval 能保留戰術文件的上下文，避免因 chunking 過度而失去語境。  
- 結合兩者能兼顧「精準度」與「上下文完整性」。  

## 後果
- 系統能更好處理軍事戰術文本中的專有名詞與上下文依賴。  
- 查詢結果更具可解釋性，避免 hallucination。  
- 計算成本較高，需在效能與準確度之間取得平衡。  

## 狀態
Accepted
